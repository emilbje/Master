1. Side View Pitch Profile

Pitch deltas are interpolated along the measured sensor span. With a factory CSV loaded, the unloaded underside and load envelope come from the measured factory bend test.

2. Top View Bending Heat

Heat is based on the pitch gradient along the ski. The view is cut at IMU0 and IMU7, so it only shows the measured span.

3. 3D Torsion From Roll

Roll deltas twist the measured sensor span only. Pitch bending is not applied in this plot, so the centerline stays at the neutral ski profile. Drag inside the plot to rotate the view.

4. Side View Torsion Heat

Heat is based on the roll gradient along the measured sensor span. Thickness follows the measured IMU values and is mildly enlarged here so the color field is easier to read.

5. Factory Bend Profiles

Load the factory CSV to draw one measured underside profile for each load. The unloaded ski is drawn translucent behind each loaded profile.

No factory CSV loaded.

6. Field Load Calibration

Mark one interval where you stand on the ski and one interval where you stomp it down. The IMU signal is then mapped to load, and the side view uses interpolated factory bend profiles through the whole test.

Fill in both intervals to enable field calibration. Leave them blank to use the fallback factory target calibration.
