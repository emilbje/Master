# Master Thesis Code

This repository contains supplementary code developed for a master's thesis project on measuring, logging, analysing and visualising deformation in cross-country skis using sensor-based prototypes.

The code was developed as part of the master's thesis work by Emil Bjerkvoll and Åse Helgeland Pedersen. The repository includes microcontroller code, Python scripts and browser-based visualisation tools used throughout the development and testing of the prototype system.
